# BMI_Predict_on_Flask

**Context**

Body mass index is a value derived from the mass and height of a person. The BMI is defined as the body mass divided by the square of the body height, and is expressed in units of kg/mÂ², resulting from mass in kilograms and height in metres.

**Content**

The dataset contains information about gender, height, weight and BMI index of individuals

Gender : Male / Female

Height : Number (cm)

Weight : Number (Kg)

**Index**

0 - Extremely Weak

1 - Weak

2 - Normal

3 - Overweight

4 - Obesity

5 - Extreme Obesity
